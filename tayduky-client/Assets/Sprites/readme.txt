HƯỚNG DẪN BUNG FILE ĐỒ HỌA GỐC CỦA GAME:

1. Tìm và tải file game "Tây Du Ký Mobile" hoặc "Tây Du Ký: Hỏa Diệm Sơn" định dạng .jar từ các trang web lưu trữ game Java cũ.
2. Đổi tên phần mở rộng của file từ `.jar` thành `.zip`.
3. Giải nén file zip bằng WinRAR, 7-Zip hoặc công cụ giải nén của hệ điều hành.
4. Sao chép toàn bộ các thư mục hình ảnh (thường chứa các file ảnh .png, .gif) vào thư mục này để Unity có thể nạp sprite hoạt ảnh.
